
******************
* MARTY THE MOLE *
******************

A game by Mercerenies, Alicia, and Cora.

Thank you for playing! The controls will be explained in-game, but
here they are for your reference.

WASD / Arrow Keys - Move Marty, navigate menu
SPACE / ENTER - Select menu item
BACKSPACE - Undo the last action
ESCAPE - Restart level
F6 - Toggle full screen
